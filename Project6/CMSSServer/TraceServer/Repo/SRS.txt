- Chức năng giám sát thư mục:
+ Theo dõi các thay đổi của một thư mục được cài đặt.
+ Có thể ghi nhận các thay đổi như: Thêm một file, xóa một file, đổi tên file, Chỉnh sửa nội dung file.
+ Các thay đổi này sẽ được gửi tới server để xử lý thích hợp

- Chức năng đồng bộ:
+ Khi nhận được báo cáo từ client server sẽ có các hành động như sau:
 Nếu là thêm file hay chỉnh sửa, server sẽ thông báo cho client có sự thay đổi đó để client đó gửi nội dung file có sự thay đổi đó lên. Đồng thời, server sẽ gửi thông báo cho các client khác để ghi file mới thêm đó.
 Nếu là xóa file hay đổi tên, server sẽ gửi yêu cầu tương ứng tới các client khác để các client đó xóa hoặc đổi tên file tương ứng.
- Chức năng đồng bộ khi kết nối:
+ Khi có một client mới kết nối đến, client sẽ gửi yêu cầu đồng bộ dữ liệu với server. Khi đó, server sẽ gửi lần lượt các file trong thư mục của mình tới cho client để đảm bảo dữ liệu giữa client và server là đồng bộ.
- Chức năng đồng bộ khi client hoặc server tắt
+ Khi server tắt, client vẫn tiếp tục giám sát thư mục của mình, và sau khi server được bật lên, client sẽ tự kết nối tới server, và khi đó, các thay đổi mà client có được trong quá trình giám sát sẽ được gửi tới server.
+ Khi client đang gửi dữ liệu tới server mà client bị ngắt kết nối, sau khi kết nối lại, server sẽ yêu cầu client gửi lại dữ liệu trước khi bị ngắt kết nối.
